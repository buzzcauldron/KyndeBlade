# Music Sources for Kynde Blade
## Copyright-Free Music for a Melancholic Medieval RPG

This document lists copyright-free music sources suitable for Kynde Blade's melancholic, contemplative themes about aging, work, poverty, and the unresolved quest for Grace.

## Licensing Guidelines

### CC0 (Public Domain)
- **No attribution required**
- **Can use commercially**
- **Can modify**
- **Best option for commercial games**

### CC BY (Attribution)
- **Attribution required**
- **Can use commercially**
- **Can modify**
- **Include credit in game credits**

### CC BY-SA (Attribution-ShareAlike)
- **Attribution required**
- **Can use commercially**
- **Can modify**
- **Must share under same license**
- **Include credit in game credits**

### Pixabay License
- **Free for commercial use**
- **No attribution required (but appreciated)**
- **Cannot claim as original work**

## Implementation Notes

### Music Integration
1. **Create music folders:**
   - `Content/Music/Act1/`
   - `Content/Music/Act2/`
   - `Content/Music/Act3/`
   - `Content/Music/Act4/`
   - `Content/Music/Act5/`
   - `Content/Music/Ambient/`
   - `Content/Music/Boss/`

2. **Naming Convention:**
   - `MUS_Act1_FirstDream`
   - `MUS_Act2_YearsPass`
   - `MUS_Boss_Pride`
   - etc.

3. **Audio Settings:**
   - Format: OGG Vorbis (recommended for Unreal)
   - Quality: 192-320 kbps
   - Sample Rate: 44.1 kHz or 48 kHz

### Credits
Create a `CREDITS.md` file listing all music sources with:
- Track name
- Artist name
- Source URL
- License type
- Attribution text (if required)

## Recommended Workflow

1. **Research Phase:**
   - Browse Free Music Archive
   - Search Incompetech catalog
   - Listen to recommended tracks
   - Build a library of potential tracks

2. **Selection Phase:**
   - Match tracks to level themes
   - Ensure consistent mood
   - Test tracks in-game context
   - Verify licensing

3. **Integration Phase:**
   - Download tracks
   - Convert to appropriate format
   - Import into Unreal Engine
   - Set up audio cues and triggers

4. **Documentation Phase:**
   - Document all sources
   - Create credits file
   - Maintain license information
   - Track attribution requirements
