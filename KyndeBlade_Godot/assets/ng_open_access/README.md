# NGA Open Access art (optional backdrop)

The **Tiny practice loop** scene (`scenes/beginner_loop.tscn`) can show a static image behind the “gallery frame” UI.

## Source

Browse **free, open-access** images from the National Gallery of Art:

- [NGA: Free Images and Open Access](https://www.nga.gov/artworks/free-images-and-open-access)

Follow NGA’s terms for each image (attribution, non-commercial or other restrictions as stated on the artwork page).

## Add a backdrop to the Godot build

1. Download an **Open Access** image you’re allowed to use (respect the work’s license on the NGA page).
2. Save it as one of:

   - `assets/ng_open_access/backdrop.png`
   - `assets/ng_open_access/backdrop.jpg` (or `.jpeg`)

3. Run the project → **Main menu** → **Tiny practice loop**.

The loop does not ship a specific NGA image in-repo; you choose the work and comply with its terms.
