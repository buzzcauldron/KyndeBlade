# Cult-ship bar — shippable soul (Godot + project)

This doc turns **parity-and-port discipline** into **player-facing identity**. It is the bar for “quirky, well considered” work that can earn a **cult** audience—not just a clean repo.

## North star

**Ship one irreducible experience** (≈30–90 minutes) that could stand alone while the full RPG grows. The Unity port and TDAD waves **serve** that artifact; they do not replace **authorship**.

## Three pillars (what players actually share)

| Pillar | Meaning | Concrete check |
|--------|---------|----------------|
| **Voice** | Language + UI + tone feel like *one* mind | A stranger quotes **one line** without naming the engine |
| **Weird rule** | One system they explain to a friend | “False **feints** so the **second** dodge…” (example) |
| **Aesthetic coherence** | Cheap-on-purpose, never accidental | Lane A/B per [`ART_DIRECTION.md`](../../ProjectArchive/UnityKyndeBlade/KyndeBlade_Unity/Assets/KyndeBlade/Docs/ART_DIRECTION.md); no generic fantasy pack mashup |

## Greatest risks in *this* build (keep visible)

1. **Godot slice = lab fidelity** — strong tests/export; thin **sensuous** craft vs Unity manuscript stack.
2. **Combat clarity before character** — telegraph is logical; cult needs **one** unmistakable read (motion, SFX, ink).
3. **Parity-oracle** can delay **Godot-only** experiments — schedule **one intentional scar** (see [`AUTHORIAL_OVERRIDE.md`](AUTHORIAL_OVERRIDE.md)).
4. **Hook not yet one GIF** — you need **one sentence + one image** only this game could claim.
