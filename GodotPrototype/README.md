# Kynde Blade Combat Prototype (Godot 4)

Minimal combat prototype to compare Godot vs Unity feel and workflow.

## Controls

- **X** — Strike (15 stamina, 15 damage)
- **Space** — Dodge (20 stamina, real-time window)
- **Shift** — Parry (25 stamina, real-time window)

## Run

1. Open project in Godot 4.x
2. Run main scene (F5)

## Features

- Turn-based player turn
- Strike / Dodge / Parry (Escapade / Ward)
- Real-time parry/dodge window
- Simple enemy turn
- Victory/Defeat

## Purpose

Compare with Unity implementation before committing to an engine.
