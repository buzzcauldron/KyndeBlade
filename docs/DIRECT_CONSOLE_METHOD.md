# Direct Console Method - No Menu Needed!

## Just Use the Console!

You don't need menus or levels. Just use the console:

### Step 1: Open Console
- **Press `** (backtick/tilde key)
- Usually located above the Tab key
- On some keyboards it's Shift+` or next to the 1 key

### Step 2: If Console Opens
- **Type**: `SpawnTestCharacters`
- **Press Enter**
- Characters spawn!

### Step 3: If Console Doesn't Open
- Try **F1** or **F2**
- Or **Ctrl+`**
- Or look for a "Console" button in any visible toolbar

## What Keyboard Do You Have?

The backtick key (`) might be in different places:
- Above Tab
- Next to the 1 key
- Shift+` (if it's on the same key as ~)
- Sometimes it's the key with ~ on it
