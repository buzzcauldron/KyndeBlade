# No Menu Bar? Here's How to Fix It

## The Menu Bar Might Be Hidden

### Try These:

1. **Press Alt** - Sometimes this shows/hides the menu bar
2. **Right-click** in the top area of the editor window
3. **Look for a small arrow** or toggle button at the very top
4. **Try F10** - Some editors toggle menu bar with this

## Use Keyboard Shortcuts Instead

You don't need the menu bar! Use these shortcuts:

### Create New Level:
- **Ctrl+N** (create new level)

### Open Console (Most Important!):
- **Press `** (backtick key, above Tab)
- This opens the console directly
- Then type: `SpawnTestCharacters`
