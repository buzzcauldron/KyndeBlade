# How to Create/Open a Level

## Method 1: Create a New Level

1. **File → New Level** (top menu bar)
2. Choose one:
   - **Empty Level** (recommended - just empty space)
   - **Basic** (has a floor)
   - **Open World** (large landscape)
3. Click **Create**
4. The level will open automatically

## Method 2: Open Existing Level

1. **File → Open Level** (top menu bar)
2. Browse to find a level file (`.umap` files)
3. Select it and click **Open**

## Method 3: Use Content Browser

1. **Open Content Browser** (bottom panel, or Window → Content Browser)
2. **Navigate** to find level files (usually in `/Content/` folder)
3. **Double-click** a `.umap` file to open it

## Method 4: Start with Default Level

When the editor opens, it usually loads a default level automatically. You can use that!

## Quick Steps:

1. **File → New Level → Empty Level**
2. Click **Create**
3. You now have a level open!

## Then Use Console Command:

Once you have a level open:

1. **Press Play**
2. **Press `** (backtick) to open console
3. **Type**: `SpawnTestCharacters`
4. **Press Enter**

That's it! You don't need anything in the level - the console command will spawn characters anywhere.

## If File Menu Doesn't Work:

Try:
- **Ctrl+N** (create new level)
- **Ctrl+O** (open level)
- Or look for "Level" in the top menu bar

Let me know if you can see the File menu or if there's a different issue!
