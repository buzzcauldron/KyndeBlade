# Actually, You Don't Need a Level!

## Good News: Console Command Works Without a Level!

The `SpawnTestCharacters` console command will work even if you're just viewing the default editor view. 

## Try This:

1. **Just press Play** (even if no level is open)
2. **Press `** (backtick) to open console
3. **Type**: `SpawnTestCharacters`
4. **Press Enter**

The characters will spawn in the default space!

## But If You Want to Create a Level:

### Easiest Way:
1. Look at the **top menu bar**
2. Click **File**
3. Click **New Level**
4. Select **Empty Level**
5. Click **Create**

### Keyboard Shortcut:
- **Ctrl+N** (create new level)
