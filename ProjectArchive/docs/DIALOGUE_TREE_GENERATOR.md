# Dialogue Tree Generator

Baldur's Gate–style complex dialogue trees for KyndeBlade, autogenerated from poem characters and game triggers.

## Overview

| Element | Source |
|---------|--------|
| **Structure** | Baldur's Gate 3 — branched trees, conditional options, consequences |
| **Characters** | Piers Plowman, Gawain, Sir Orfeo — Wille, Piers, Lady Mede, Green Knight, etc. |
| **Triggers** | Game philosophy — HasVisited, PovertyLevel, EldeHits, GreenKnightRandom, OrfeoTriggered |

## Usage

1. **Generate default tree:** KyndeBlade > Generate Default Dialogue Tree (Poems + Triggers)
   - Creates `Assets/Resources/Data/DialogueTrees/DefaultDialogueTree.asset`
   - Creates `Choice_green_chapel_intro.asset`, `Choice_boundary_tree.asset`, etc.

2. **Customize:** Edit the DialogueTreeDefinition, add nodes, then KyndeBlade > Generate Dialogue Tree

3. **Wire to locations:** Assign generated DialogueChoiceBeat to `LocationNode.PreCombatChoiceBeat` (e.g. Green Chapel, Boundary Tree)

## Data Structures

### DialogueTreeDefinition

- **Nodes[]** — each node has NodeId, Text, SpeakerName, Choices[]
- **ChoiceDef** — Text, IsCorrectChoice, TransitionToLocationId, AssociatedSin, NextNodeId (chaining), Condition, Consequence

### Conditions (Baldur's Gate–style)

Checked before choice is shown. Future: runtime filter.

| Condition | Param | Meaning |
|-----------|-------|---------|
| HasVisited | StringParam | Only if player has visited location |
| HasNotVisited | StringParam | Only if not visited |
| PovertyLevelAtLeast | IntParam | Poverty >= N |
| EldeHitsAtLeast | IntParam | Elde hits >= N |
| GreenKnightWillAppear | — | Green Knight in random pool |
| OrfeoTriggered | — | Otherworld path opened |
| HasEverHadHunger | — | Hunger scar on party |

### Consequences (game triggers)

Applied when choice is selected.

| Consequence | Effect |
|-------------|--------|
| SetGreenKnightRandom | SaveManager.GreenKnightWillAppearRandomly |
| SetOrfeoTriggered | SaveManager.OrfeoOtherworldTriggered |
| TransitionToLocation | WorldMapManager.TransitionTo |
| StartSinMiniboss | Combat with sin enemy |

## Poem Sources

- **Green Chapel** — Sir Gawain and the Green Knight, beheading game
- **Boundary Tree** — Sir Orfeo, fairy realm threshold
- **Piers / Do-Wel** — Piers Plowman, Passus VIII
- **Lady Mede** — Piers Plowman, Lady Meed temptation

## Chained Dialogue

Choices can have `NextNodeId` → generates `NextDialogueBeat` reference. NarrativeManager shows the next dialogue before invoking the final callback. Example: Lady Mede temptation → Piers approves.
