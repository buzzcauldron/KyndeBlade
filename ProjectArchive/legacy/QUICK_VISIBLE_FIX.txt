QUICK FIX - Make Characters Visible
===================================

1. Right-click in Content Browser → Blueprint Class
2. Search "Knight Character" → Create "BP_PlayerKnight"
3. Open BP_PlayerKnight
4. Select "Skeletal Mesh Component" in Components
5. In Details, find "Skeletal Mesh" dropdown
6. Select a mesh (use Starter Content "Mannequin" if available)
7. Repeat for Mage, False, Lady Mede, Wrath

8. Update spawner:
   - Project Settings → Game → Default Game Mode
   - Set classes to your Blueprints

9. Press Play - characters should be visible!

The spheres are just debug markers - characters are there, just need meshes!
