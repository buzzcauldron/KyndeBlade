# Legacy — Unreal Engine 5.7

This folder contains the original Kynde Blade project built for Unreal Engine 5.7.

The active project is now **Kynde Blade 16 Bit** in Unity (see repo root). This Unreal version is archived for reference.

## Build

```bash
./build_5.7.sh [path/to/UnrealEngine]
```

## Contents

- `Source/` — C++ source (Characters, Combat, Game, UI, Input)
- `Config/` — Engine config
- `Content/` — Unreal assets
- `KyndeBlade.uproject` — Project file
- `unreal-5.7/` — Merged kyndeblade-5.7 content (Archer, Rogue, CombatPlayerController, CombatAIController, etc.)
